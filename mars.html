Mars Weather App


Using two NASA APIs, I created a Mars Weather App. The first API is called Insight and it gives me the weather in Mars, which I display in the boxes at the bottom and on the lower left. The second API is called Mars Rover Photos and it gives me images from Rovers in Mars, which I use in the background. The user can interact with the website when they hover over the image. They can type in a date and search of images taken on that soecific day, which will then turn into the new background image. If there are no images registered under that day, an error message shows up and the user can try another date.
